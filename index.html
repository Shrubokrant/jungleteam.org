Octoputs — 3D Drag & Drop

Runtime Editor

Drag and drop in Unity sounds simple, until you need to ensure it can coordinate inventories, automation, have collision avoidance, smooth game feel, or proper snapping behaviour. Octoputs handles all of it with a composable modifier pipeline.

  • 33 stackable modifiers
  • Inventory, automation, card game, 3D drag and drop, VR, …
  • 8 targeting modes
  • 5 layout strategies
  • Attachment & transfer system
  • 7 example scenes
  • Physics & collision avoidance
  • Works with 2D and 3D

One Modular Framework, all Aspects of Game Dev

Every plugin above is powered by Jungle, a modular foundation split into small packages. When you buy a plugin, only the modules it needs are installed. Buy a second plugin? Shared modules are already there. No duplicates, no conflicts.

Values, events, conditions, processes, actions: all the building blocks necessary for making games are available, only if you want them. Full source, (close to) zero runtime allocation.
